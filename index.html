DNS Record Types

Understanding different DNS record types and when to use them.

A
A Record Maps to IPv4 address (e.g., 192.168.1.1)
AAAA
AAAA Record Maps to IPv6 address (e.g., 2001:db8::1)
CNAME
CNAME Record Alias to another domain (e.g., example.com)
URL
URL Redirect HTTP redirect to any URL

Troubleshooting

Common issues and their solutions.

Q: Subdomain not resolving?

Wait for DNS propagation (up to 48h). Clear your DNS cache.

Q: Can't claim more subdomains?

Free users are limited to 5 subdomains. Delete unused ones.

Q: Wrong IP address showing?

Edit your subdomain and update the destination.

API Documentation

Integrate SubHub into your applications

GET
/api/check
Check subdomain availability
curl https://subhub.dev/api/check?subdomain=myapp&domain=subhub.dev
POST
/api/claim
Claim a subdomain
curl -X POST https://subhub.dev/api/claim \
-H "Content-Type: application/json" \
-d '{"subdomain":"myapp","domain":"subhub.dev","type":"A","destination":"192.168.1.1"}'
GET
/api/subdomains
List your subdomains
curl https://subhub.dev/api/subdomains -H "Authorization: Bearer YOUR_TOKEN"
